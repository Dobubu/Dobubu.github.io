在mac環境下安裝nvm過程

主要紀錄，我在mac上安裝nvm遇到的問題，和最後怎麼解決，讓我們可以繼續開心做下去

安装 nvm

這邊你可以在 github nvm 上看到

接著出現下方資訊
這表示你的系統中沒有 .bash_profile 這個檔案

接著輸入 nvm 查看，會看到下面這個東西。失敗 :(

讓我們來好好解決它！！！

先開啟你的資料夾（這是我的）

然後使用 shift + cmd + .
來開啟隱藏檔案，就會看到很多的隱藏檔（再使用一次 shift + cmd + . 就可以隱藏 隱藏檔案）
這邊你可以使用 touch .bash_profile 來新增檔案

這邊截圖已經是我增加 .bash_profile 檔案後的畫面了 ～

接著再開啟新的terminal，輸入一開始的安裝指令

接著出現下列資訊，代表你成功了 :)

接著輸入 nvm 查看是否安裝成功

會列出一些資訊（如下圖）
這邊就代表安裝成功，恭喜大家～

若還是不行，開啟剛剛新增的 .bash_profile
並添加代碼 source ~/.bashrc
存擋後，再跑一次步驟
