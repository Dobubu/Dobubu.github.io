JavaScript-Patterns-2012-介紹[1-2]

頁碼 p.16 ~ p.22

for迴圈（重複陣列）

可遍歷整個陣列或類似陣列的物件

迴圈模式的變形。優化效能

  • 可少用一個變數（這邊指 max）👍
  • 遞減至0，通常比較快 👍

for-in迴圈（重複物件）

用來歷整整個非陣列物件，又稱「列舉 enumeration」

  • 使用 hasOwnProperty() 過濾掉來自原型鏈的屬性

  • 也可使用，Object.prototype呼叫 hasOwnProperty()

    • 避免命名衝突
    • 若嫌這句話太長，可以用一個區域變數快取它

不要擴充內建型別的原型

switch

下面模式提升可讀性和穩健性

  • 將每個 case 與 switch 並排

  • 每個 case 內的程式碼縮排

  • 用 break; 結束每個 case

  • 用 default; 結束 switch，即使沒有任何一個 case 符合
