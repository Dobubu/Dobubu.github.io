for迴圈（重複陣列

  • 可少用一個變數（這邊指 max）👍
  • 遞減至0，通常比較快 👍

for-in迴圈（重複物件

用來歷整整個非陣列物件，又稱「列舉 enumeration」

  • 使用 hasOwnProperty() 過濾掉來自原型鏈的屬性

  • 也可使用，Object.prototype呼叫 hasOwnProperty()

    • 避免命名衝突
    • 若嫌這句話太長，可以用一個區域變數快取它
